Sobre Rust

Rust é uma linguagem de programação compilada que pode ser usada em diversos segmentos. Desde web development até embarcados. Sendo uma linguagem prática, que mistura elementos de alto e baixo nível empoderando a todos a criar código rápido, eficiente e seguro.

Com um sistema de tipos forte e estático, Rust ajuda a garantir que seus erros vão acontecer na compilação, e não em execução. Com isso, com rust é possível criar sistemas confiáveis, com baixo footprint de memória.

Apesar do crescimento da linguagem, Rust ainda é pobre em grupos de meet-up e conferências principalmente no Brasil. Por isso criamos o RustCWB.
